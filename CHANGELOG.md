# Histórico de Atualizações

## [v1.1.0] - 2026-02-11
### Segurança
- **Integração CSRF Completa:** Proteção contra Cross-Site Request Forgery em todos os 46 formulários do sistema.
- **Security Headers:** Adicionados headers HTTP de segurança (HSTS, XSS Protection, etc).
- **Rate Limiting:** Novo sistema de proteção contra força bruta no login.
- **Validação de Senhas:** Regras mais rigorosas para criação de senhas.
- **ReCAPTCHA:** Integração flexível com Google reCAPTCHA v2/v3.
- **Gestão de Sessão:** Cookies seguros (HttpOnly, Secure, SameSite).

### Admin
- **Nova Página de Atualizações:** Visualização deste changelog diretamente no painel.
- **Correções de Menu:** Adicionados links faltantes no menu lateral.

### Lançamento Inicial
