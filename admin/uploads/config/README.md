# Diretório de Uploads de Configuração

Este diretório armazena as imagens de configuração do sistema:

- **logo.{ext}** - Logo do site (aparece na sidebar e no perfil)
- **favicon.{ext}** - Favicon do site (aparece na aba do navegador)
- **capa.{ext}** - Imagem de capa do site (hero/banner)

Os arquivos são sempre sobrescritos com nomes fixos (sem timestamp).

Extensões suportadas:
- Logo: jpg, jpeg, png, gif, webp, svg
- Favicon: ico, png, jpg, jpeg
- Capa: jpg, jpeg, png, gif, webp
